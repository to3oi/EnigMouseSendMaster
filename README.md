# EnigMouseSendMaster
ゲームEnigMouseで使用する入力ソフトです。

物体検出をした結果をゲーム本体へ送信します。


---
このリポジトリは以下のリポジトリの内容を引き継いで作成されています。

[https://github.com/to3oi/KinectImageConvertSender](https://github.com/to3oi/KinectImageConvertSender)
